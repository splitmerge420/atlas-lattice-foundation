## Contributing

All white papers must include YAML frontmatter with the following fields:
- `title` — Full paper title
- `version` — Semantic version
- `date` — Publication date (YYYY-MM-DD)
- `author` — Authoring entity and council role
- `classification` — Paper classification level
- `sphere_tags` — Relevant sphere identifiers
- `aluminum_layer` — Constitutional layer designation
- `council_status` — Approval status
- `cross_refs` — Related documents in other repos
