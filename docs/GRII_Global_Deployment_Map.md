## The 5 Success Factors for Rapid Adoption

These countries share a pattern. If a nation has 3+ of these, GRII can scale in less than 24 months:

| Factor | Why It Matters | Example |
|--------|---------------|---------|
| **1. Policy Tailwind** | Statutory mandates or budget allocations create pull | India's CAQM stubble direction; Indonesia's 34-project announcement |
| **2. Proven Pilot** | Local proof-of-concept reduces perceived risk | Banas Dairy (India); Sistema Biobolsa (Kenya); Green Hero (Vietnam) |
| **3. Waste Stream Abundance** | Feedstock availability = economic viability | Indonesia's urban waste; Vietnam's 60% organic municipal stream |
| **4. Import Vulnerability** | Pain creates urgency for sovereignty | SE Asia's 75-100% phosphorus import dependency |
| **5. Partnership Infrastructure** | Existing networks accelerate deployment | IGL's 22 CBG partners (India); Denmark-Vietnam green strategic partnership |

## Strategic Phasing: Anchor Corridors to Systemic Integration

### Phase 1: Anchor Corridors (0-12 months)

Focus on countries with Policy + Pilot + Pain:

- **India**: Punjab-Haryana stubble corridor + Banas replication (100 DISH-printed nodes)
- **Indonesia**: Urban WtE nodes in Jakarta, Surabaya, Medan (integrate BAETA + AI)

### Phase 2: Partnership Multipliers (12-24 months)

Leverage existing bilateral frameworks:

- **Vietnam + Denmark**: Scale Green Hero to Mekong Delta
- **Malaysia + Singapore**: Cross-border waste-to-value network

### Phase 3: Systemic Integration (24-36 months)

Connect corridors via Rosetta Layer:

- Fertilizer sovereignty metrics shared across India/SE Asia/Africa
- Carbon credit interoperability via Optical Ledger
